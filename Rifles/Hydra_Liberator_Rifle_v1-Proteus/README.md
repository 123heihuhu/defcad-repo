# Hydra Liberator Rifle conversion 

####VERSION 
* 1.0

#### Note:
* Uses standard Liberator v1.1 parts.


#### Instructions
1. Print all Liberator v1.1 parts except the barrel and grip. Instead, use barrel1 and modifiedgrip contained.
2. Attached grip as usual and barrel1 as usual. Use Weld-on or similar substance to combine the three barrel parts as well as the two stock parts to the back of the grip.
